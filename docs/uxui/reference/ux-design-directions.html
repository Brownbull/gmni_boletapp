Design Direction v2 - Key Changes

1. Always-Visible Scan Button

  • - Center FAB (Floating Action Button) in bottom nav
  • - Document + scan icon (receipt scanning)
  • - Gradient accent color to stand out
  • - Critical differentiator from budgeting-only apps

2. Collapsible Breadcrumbs

  • - Two compact buttons side-by-side (When | What)
  • - Shows current level only (e.g., "November" or "Groceries")
  • - Tap to expand full navigation dropdown
  • - All ancestor levels visible and tappable when expanded

3. Real Pie Chart

  • - Actual pie segments with real percentages
  • - Each segment colored by category
  • - SVG-based for crisp rendering
  • - Category legend below chart

4. Stacked Bar Chart

  • - Each bar shows total for that period
  • - Stacked segments show category breakdown
  • - Same colors as pie chart for consistency
  • - Category legend shown in both views
