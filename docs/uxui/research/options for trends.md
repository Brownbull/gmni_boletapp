**Key Design Elements:**

1. **Dotted baseline** = Last period's spending (reference point)
2. **Solid line** = This period's cumulative spending
3. **Green** = Line stays below baseline (spending less)
4. **Red** = Line goes above baseline (spending more)
5. **Gray** = Line near baseline (similar spending)
6. **Right side** = Total + percentage change with emoji indicator

**The Mental Model:**
> "The dotted line is where I was. If I stay below it, I'm winning."

**SVG Sparkline Component:**
```typescript
interface SparklineData {
  current: number[];    // Cumulative daily totals
  baseline: number[];   // Last period cumulative
}

function renderSparkline(data: SparklineData, status: SpendingStatus): SVGElement {
  const width = 120;
  const height = 40;
  const maxValue = Math.max(...data.current, ...data.baseline);
  
  // Normalize to SVG coordinates
  const normalize = (val: number) => height - (val / maxValue) * height;
  
  // Generate path for baseline (dotted)
  const baselinePath = data.baseline
    .map((v, i) => `${i === 0 ? 'M' : 'L'} ${i * (width / (data.baseline.length - 1))} ${normalize(v)}`)
    .join(' ');
  
  // Generate path for current (solid)
  const currentPath = data.current
    .map((v, i) => `${i === 0 ? 'M' : 'L'} ${i * (width / (data.current.length - 1))} ${normalize(v)}`)
    .join(' ');
  
  return `
    <svg width="${width}" height="${height}" viewBox="0 0 ${width} ${height}">
      <!-- Baseline (dotted) -->
      <path d="${baselinePath}" 
            fill="none" 
            stroke="#9CA3AF" 
            stroke-width="1.5" 
            stroke-dasharray="4 2" />
      
      <!-- Current period (solid with fill) -->
      <path d="${currentPath} L ${width} ${height} L 0 ${height} Z" 
            fill="${statusStyles[status].fillColor}" />
      <path d="${currentPath}" 
            fill="none" 
            stroke="${statusStyles[status].lineColor}" 
            stroke-width="2" />
    </svg>
  `;
}
```
