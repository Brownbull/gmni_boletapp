Semantic Colors - Graphics & Indicators

Colors for trend indicators, charts, progress bars, and other graphical elements. Each theme has harmonized colors that feel cohesive with the overall aesthetic.

Trend Indicators

Used for spending changes: positive = spending decreased (good), negative = spending increased (bad)

↓
Positive / Down
Spending decreased
-15% vs last week $50 saved
--positive-primary | --positive-bg | --positive-border
↑
Negative / Up
Spending increased
+23% vs last week Over budget
--negative-primary | --negative-bg | --negative-border
→
Neutral
No significant change
~0% change On track
--neutral-primary | --neutral-bg | --neutral-border
⚠
Warning
Approaching limit
85% of budget Caution
--warning-primary | --warning-bg | --warning-border

CSS Variables Reference

/* Positive (spending down - good) */ --positive-primary /* Main text/icon color */ --positive-secondary /* Secondary/hover color */ --positive-bg /* Background fill */ --positive-border /* Border/outline color */ /* Negative (spending up - bad) */ --negative-primary --negative-secondary --negative-bg --negative-border /* Neutral (no change) */ --neutral-primary --neutral-secondary --neutral-bg --neutral-border /* Warning (approaching limits) */ --warning-primary --warning-secondary --warning-bg --warning-border /* Chart palette (6 harmonized colors) */ --chart-1 through --chart-6
