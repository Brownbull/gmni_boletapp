Core Scan Flows
Flujo 1: Primera Boleta
New user onboarding: login, empty dashboard, first scan, first save, celebration.
First-time user
Flujo 2: QuickSave Diario
Daily habit: scan, confidence check, QuickSave (3 taps) or edit path.
Daily use
Flujo 3: Lote de Boletas
Batch capture: mode selector, multi-photo, credit check, batch review, save all.
Batch scan
Flujo 4: Estado de Cuenta
Statement scan: consent, upload PDF, async processing, matching, reconciliation.
Epic 18
Intelligence & Recovery
Flujo 6: Aprendizaje → Confianza
Progressive learning: manual correction, auto-categorize, trust merchant, auto-save.
Multi-session
Flujo 7: Créditos Agotados
Credit depletion: low balance, blocked scan, wait or purchase recovery paths.
Edge case
Flujo 8: Recuperación de Error
Scan failure: network error, retry or cancel with credit refund transparency.
Error handling
Flujo 9: Offline → Reconexión
PWA offline: queued scans, cached data, auto-sync on reconnect.
PWA
Exploration & Export
Flujo 5: Gastos Compartidos
Group sharing: context switch, auto-copy, batch add, admin/invite. 3 sub-paths.

Flujo 10: Análisis Profundo
Trends deep dive: donut, sankey, treemap, drill-down, cross-view to History.
Analytics
Flujo 11: Reportes & PDF
Reports: accordion, detail overlay, inline donuts, transaction pill, PDF export.
Reports
Flujo 12: Exportar Datos
4 export entry points: Settings, History, Trends, Items. CSV/PDF formats.
Export
