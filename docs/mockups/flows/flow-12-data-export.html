Seguridad
Sanitizado
"Tengo mis datos, puedo usarlos donde quiera."
Todas las exportaciones pasan por sanitizeCSV() antes de generar el archivo. Se elimina BOM, se sanitizan formulas, y se valida encoding UTF-8.
🛡️
Seguridad: CSV Injection Prevention
Los exports CSV sanitizan formulas potencialmente peligrosas. Caracteres =, +, -, @ al inicio de celdas son prefijados con apostrofe para prevenir inyeccion cuando el archivo se abre en Excel o Google Sheets.
