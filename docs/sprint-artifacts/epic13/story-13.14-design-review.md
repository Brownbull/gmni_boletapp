## User Story

As a **product owner**,
I want **a comprehensive design review before implementation**,
So that **all mockups are consistent, complete, and approved before Epic 14 begins**.

---

- [ ] **AC #1:** All mockups reviewed for visual consistency
- [ ] **AC #2:** Animation specifications validated against mockups
- [ ] **AC #3:** Voice & tone guidelines applied throughout all mockups
- [ ] **AC #4:** User (Gabe) approval obtained

## Review Checklist

### Visual Consistency
- [ ] Color palette consistent across all mockups
- [ ] Typography scale followed
- [ ] Spacing system applied uniformly
- [ ] Icon style consistent (Lucide, 24px)
- [ ] Component patterns reused appropriately

### Animation Alignment
- [ ] All animation notes reference Motion Design Spec
- [ ] Timing specifications match spec
- [ ] Breathing effects use correct parameters
- [ ] Settings exception documented

### Voice & Tone
- [ ] All copy follows voice principles
- [ ] Chilean adaptations applied
- [ ] Rosa-friendly alternatives included
- [ ] No judgmental language

### Persona Validation
- [ ] María: Home Dashboard validates
- [ ] Diego: Goals GPS validates
- [ ] Rosa: Simple formats included
- [ ] Tomás: Airlock validates

### Completeness
- [ ] All 8 mockup stories complete
- [ ] All states designed (default, empty, error)
- [ ] Mobile viewport (375px) verified
- [ ] Dark mode considered
